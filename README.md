# FDGA
Short and simple 2D metroidvania. Explore the map, unlock new abilities, fight enemies, discover secrets, and avoid traps!

Features:
* Combat system
* Unique abilities
* Hidden puzzles 
* Map split into 6 regions
* Progress tracking

Play now: https://dewle.itch.io/fdga.

Made in Unity 2022.3.37f1.
